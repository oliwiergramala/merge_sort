# Merge Sort Project

## About the project

This project was created for proseminar classes.\
The goal of the project is to demonstrate both **recursive** and
**iterative** implementations of the **Merge Sort algorithm**.

The algorithm is implemented in three different programming languages: -
C - Go - Python

## Implementations

Each language contains two versions of the algorithm: - **Recursive
Merge Sort** - **Iterative Merge Sort**

## Algorithm description

Merge Sort is a **divide and conquer** sorting algorithm. It works
by: 1. Dividing the array into smaller parts 2. Sorting the smaller
parts 3. Merging the sorted parts back together
